# Adarsh K — Portfolio Website

A dark minimalist personal portfolio website for Adarsh K, final-year B.Tech student in Computer Science & AI.

## ✨ Features
- 🎬 **FX Slider** — Full-screen animated sections reveal on scroll
- 📚 **3D Book Projects** — Hover over books to reveal project details
- 🔐 **Puzzle Gate** — Solve 3 riddles to unlock full portfolio vault
- 🎨 **Dark Minimalist Design** — Premium dark theme with glassmorphism
- 🎉 **Confetti Unlock** — Celebrate puzzle completion with confetti
- 📱 **Fully Responsive** — Works on all devices

## 🛠️ Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Space Grotesk + Inter)
- CSS Custom Properties & Animations
- Intersection Observer API

## 🚀 Deployment
